# seegfui-print-assets

Öffentliche Druck-Assets (PNG/SVG) der Marke Seegfui, ausgeliefert über jsDelivr
für die Gelato-Druckanbindung. 1:1 aus `djmana78/seegfui` → `designs/` gespiegelt
(nur `*_transparent.png` und `*_vektor.svg`), auf Tags gepinnt. Nicht von Hand
bearbeiten — Updates über `scripts/assets/publish-jsdelivr.sh` im Code-Repo.
